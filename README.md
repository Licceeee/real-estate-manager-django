# real-estate-django
Real estate manager

clone

```git clone git@github.com:Licceeee/real-estate-manager-django.git```

change dir

```cd real-estate-django```

create venv

```virtualenv -p python3 venv```

activate venv

```source venv/bin/activate```

install dependencies

```pip install -r req.txt```
